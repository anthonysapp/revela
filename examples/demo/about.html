About Revela

Revela is a powerful TypeScript library designed for creating smooth scroll-based animations and page transitions. It integrates seamlessly with modern web frameworks and provides a clean API for managing section reveals.

Key Features

Built with performance in mind, Revela uses hardware-accelerated animations, respects user motion preferences, and provides excellent TypeScript support for better developer experience.

Easy Integration

Whether you're building a simple landing page or a complex web application, Revela's modular architecture makes it easy to integrate scroll animations into your project.
